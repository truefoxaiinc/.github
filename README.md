# Truefox AI Inc community standards

This repository contains shared contribution, security, support, issue, pull-request, dependency and CI templates for Truefox AI Inc repositories.

The visible account profile is maintained in [`truefoxaiinc/truefoxaiinc`](https://github.com/truefoxaiinc/truefoxaiinc).

Project-specific files take precedence when a repository requires stricter controls.
